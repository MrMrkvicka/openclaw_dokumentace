Instalace a konfigurace OpenClaw s Ollama na GNU/Linux

Škola: Obchodní akademie OAUH

Autoři: Matěj Šmíd, Tomáš Ondra

Datum: 8. června 2026

Operační systém: Debian GNU/Linux

Požadavky

  • Operační systém Debian GNU/Linux
  • Alespoň 8 GB RAM (doporučeno 16 GB a více)
  • Alespoň 5 GB volného místa na disku
  • Připojení k internetu (pro instalaci)

Postup instalace

Krok 1 – Instalace curl

Na čisté instalaci Debianu nemusí být nástroj curl k dispozici. Nainstalujeme ho příkazem:

sudo apt update && sudo apt install curl -y

Krok 2 – Instalace OpenClaw

OpenClaw nainstalujeme pomocí oficiálního instalačního skriptu:

curl -fsSL https://openclaw.ai/install.sh | bash
Při průvodci nastavte: model provider → Ollama, výchozí model → Ollama/Demo, komunikační kanál → Skip for now, web search → Skip for now.

Krok 3 – Generování gateway tokenu

Pro přístup do webového rozhraní je potřeba vygenerovat autentizační token, pokud se nevygeneroval automaticky:

openclaw doctor --generate-gateway-token

Po vygenerování otevřete prohlížeč na adrese http://127.0.0.1:18789/ a token vložte do přihlašovacího formuláře.

Token je citlivý údaj – nesdílejte ho veřejně ani ho neukládejte do GitHub repozitáře.

Krok 4 – Instalace Ollama

Ollama slouží ke stahování a spouštění jazykových modelů lokálně:

curl -fsSL https://ollama.com/install.sh | sh
Instalátor může zobrazit chybu „useradd: not found" – tato chyba je nezávažná a funkčnost Ollama neovlivní.

Krok 5 – Spuštění Ollama serveru

Ollama server musíme spustit na pozadí, aby byl dostupný pro OpenClaw:

ollama serve &
Znak & zajistí spuštění na pozadí, terminál zůstane volný pro další příkazy.

Krok 6 – Stažení lokálního modelu

Stáhneme jazykový model llama3.2:3b (cca 2 GB, vhodný pro počítače s 8+ GB RAM):

ollama pull llama3.2:3b

Úspěšné stažení ověříme příkazem:

ollama list

Krok 7 – Konfigurace modelu v OpenClaw

Otevřeme soubor ~/.openclaw/openclaw.json a upravíme sekci agents.defaults.models:

"models": {
  "ollama/Demo": {},
  "ollama/llama3.2:3b": {}
}

A nastavíme primární model agenta:

"model": { "primary": "ollama/llama3.2:3b" }

Krok 8 – Spuštění po restartu

OpenClaw se spouští automaticky přes systemd – stačí otevřít prohlížeč na http://127.0.0.1:18789/. Ollama je třeba spustit ručně:

ollama serve &
